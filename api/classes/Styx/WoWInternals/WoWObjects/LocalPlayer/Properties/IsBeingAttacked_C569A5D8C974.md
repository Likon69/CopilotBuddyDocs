# LocalPlayer.IsBeingAttacked Property

Returns true if the player is currently being attacked by something. This is a lightweight approximation of HonorBuddy's logic, using threat status when available and falling back to "in combat".


## Property Value

true if is being attacked; otherwise, false.
