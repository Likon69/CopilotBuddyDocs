# BotBase.RequiresProfile Property

Gets a value indicating whether this bot requires a profile to be loaded. If false, the bot can run without a profile (like CombatBot, LazyRaider).

## Syntax

```csharp
public virtual bool RequiresProfile { get; }
```

## Property Value

Type: bool
true if requires profile; otherwise, false.
