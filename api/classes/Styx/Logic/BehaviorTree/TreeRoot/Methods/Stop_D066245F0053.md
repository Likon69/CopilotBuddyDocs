# TreeRoot.Stop Method

Stop the bot. HB 6.2.3 pattern: ONLY sets State = Stopping. The worker thread detects the state change, finishes the current tick, then does the actual cleanup on its own thread — no race condition. NO Join() — HB 6.2.3 never joins the worker thread from Stop().
