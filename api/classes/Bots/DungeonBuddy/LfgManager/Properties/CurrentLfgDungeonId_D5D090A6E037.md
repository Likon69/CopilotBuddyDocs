# LfgManager.CurrentLfgDungeonId Property

ID du donjon LFG actuel (depuis GetPartyLFGID()). Retourne 0 si pas dans un groupe LFG. HB 4.3.4: DungeonManager utilise cette valeur pour choisir le script de donjon.
